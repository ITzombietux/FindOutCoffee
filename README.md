# 커피를 찾아서
내 주변 1km의 카페를 지도로 보여주고 커피의 후기를 한 눈에 볼 수 있는 앱

## 개발자 소개
* 만사 : 세상의 만사형통(萬事亨通)을 꿈꾸는 iOS개발자

## 활동 사항
- [x] MVVM 디자인 패턴을 적용한 개발
- [x] Firebase 실시간 데이터베이스를 통한 서버 관리
- [x] 네이버 지도, 카카오 지도 API를 사용한 UI개발
- [ ] OAuth 간편 로그인을 사용한 계정관리 및 후기 작성
